ASCII for the interface. $ascii

Agents compose fresh glyphs, frames, and texture for each UI. This page uses a sparse field—quiet motion, lots of empty space.

npx ascii-skill

Install

Runs from npm and copies SKILL.md, references/, and assets/ into your skill folders. Default targets both .claude/skills (project) and ~/.codex/skills (Codex).

npx ascii-skill

npx ascii-skill install --claude Only ./.claude/skills/ascii-skill

npx ascii-skill install --codex Only ~/.codex/skills/ascii-skill

Before publish to npm, install from the repo: npx github:arjunkshah/ascii-skill
