How the skill works

        

          

            
1

            

              
Context pick

              

                Classify the surface: hero for focal UI, feature for supportive blocks,
                ambient for bento backgrounds.
              

            

          

          

            
2

            

              
Drop the script

              

                Add assets/ascii-skill.js, mark containers with data-ascii-skill, and call
                AsciiSkill.initAll() after load (or wire manually in your framework).
              

            

          

          

            
3

            

              
Theme + motion

              

                Colors inherit from CSS variables on the same node. Mouse input biases brightness; hero/feature tilt uses light 3D
                perspective on the canvas for depth without hurting readability.
