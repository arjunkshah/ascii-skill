Optional runner

            

              ascii-skill.js is a small helper for reactive fields. It never replaces designing ASCII for the
              actual interface.
            

          

        

      


      

        
How the skill works

        

          

            
1

            

              
Read the surface

              

                Decide hero, feature, or ambient, then sketch new
                ASCII—frames, shading, word blocks—that fits this product’s theme and layout.
              

            

          

          

            
2

            

              
Ship as markup

              

                Encode it in <pre> / CSS stacks; tune mono font, color tokens, and contrast. Resize or simplify
                ornate frames on small screens if needed.
              

            

          

          

            
3

            

              
Add motion only if it fits

              

                When a live field helps, pull in ascii-skill.js, scope it to one region, and theme it with CSS
                variables—still layered under real content.
